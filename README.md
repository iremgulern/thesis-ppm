
# Judicial Workload PPM

A machine learning pipeline to predict lawsuit duration using Brazilian court data.

## Quick Start

### 1. Install Requirements
```
pip install -r requirements.txt
````

### 2. Place Data (IMPORTANT)

You must place the csv file in the data folder:
`data/TJSP-BL-event-log.csv`


### 3. Run

```
python main.py
```

## Findings & Results

Once the pipeline finishes, you can find the **results**, **tables**, and **plots** in the `reports/` folder.
